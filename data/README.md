# Data

Datasets are loaded via course-provided helper functions (`utils.py`, `load_data()`, etc.) from the Coursera lab environment.

Key datasets used across notebooks:
- **Restaurant profits** (C1_W2): 97 examples, population vs. profit
- **University admissions** (C1_W3): 100 examples, 2 exam scores → admit/reject
- **Handwritten digits** (C2_W1, C2_W2): Modified MNIST subset, 20×20 pixel images (400 features)
- **Mushroom classification** (C2_W4): 10 examples, 3 binary features → edible/poisonous
- **Server metrics** (C3_W1 Anomaly): 307 examples, 2 features (throughput, latency)
- **Image pixels** (C3_W1 K-Means): Bird image, 128×128×3 RGB values
- **MovieLens ml-latest-small** (C3_W2): 397 users, 847 movies, 25,521 ratings (0.5–5.0 scale)
- **Lunar Lander** (C3_W3): OpenAI Gym environment, 8-dim state space, 4 discrete actions
